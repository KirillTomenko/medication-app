**Почему именно эти теги:**
- `pwa`, `service-worker` — находят люди, ищущие PWA-примеры
- `google-sheets`, `google-apps-script` — популярные теги для интеграций
- `medication-tracker`, `health`, `healthcare` — тематические теги, релевантные контенту
- `vanilla-javascript`, `single-file` — привлекают разработчиков, которым интересны простые решения без зависимостей
- `reminder-app`, `offline-app` — описывают ключевые функции

GitHub рекомендует не более 20 тегов. Этот список из 13 — оптимален.
